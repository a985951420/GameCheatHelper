## 简介

**GameCheatHelper** 是一个游戏秘籍快捷输入工具，专为《魔兽争霸3》和《星际争霸1》设计。通过全局热键，您可以快速在游戏中输入秘籍，无需手动打字，提升游戏体验。

### 主要特性

- 🎮 自动检测游戏进程
- ⌨️ 全局热键快速输入秘籍
- 📝 支持自定义秘籍和热键绑定
- 🔍 秘籍搜索和分类管理
- 🌐 系统托盘最小化
- ⚙️ 可配置的输入延迟和检测间隔

## 安装说明

### 系统要求

- 操作系统: Windows 10/11 (64位)
- .NET 8.0 Runtime 或更高版本
- 支持的游戏: 魔兽争霸3、星际争霸1

### 安装步骤

1. 下载 `GameCheatHelper_v1.0.zip` 压缩包
2. 解压到任意目录（建议解压到 `C:\Program Files\GameCheatHelper`）
3. 运行 `GameCheatHelper.exe` 启动程序
4. (可选) 在设置中启用"开机自动启动"

## 快速开始

### 首次使用

1. **启动应用程序**
   - 双击 `GameCheatHelper.exe` 运行程序
   - 程序会显示主窗口并开始检测游戏

2. **启动游戏**
   - 打开《魔兽争霸3》或《星际争霸1》
   - 程序会自动检测到游戏并显示"检测到 XXX"

3. **使用热键输入秘籍**
   - 在游戏中按下预设的热键（如 F1、F2、F3）
   - 程序会自动在游戏聊天框中输入对应的秘籍代码

4. **查看秘籍列表**
   - 在主窗口的秘籍列表中查看所有可用的秘籍
   - 每个秘籍旁边显示其绑定的热键

## 功能介绍

### 主窗口

主窗口包含以下区域：

#### 1. 顶部状态栏
- 显示当前检测到的游戏状态
- "设置"按钮：打开设置窗口

#### 2. 搜索栏
- 输入关键词搜索秘籍
- 支持搜索秘籍代码、描述和分类

#### 3. 秘籍列表
显示当前游戏的所有秘籍，包含：
- **热键**: 绑定的快捷键
- **秘籍代码**: 实际输入的代码
- **描述**: 秘籍的作用说明
- **分类**: 秘籍类别（资源、战斗、地图等）
- **启用**: 勾选框，控制是否启用该秘籍

#### 4. 操作按钮
- **➕ 添加秘籍**: 添加自定义秘籍
- **✏ 编辑**: 编辑选中的秘籍
- **🗑 删除**: 删除选中的秘籍

#### 5. 底部状态栏
- 显示应用程序的实时状态消息
- 显示应用程序版本号

### 系统托盘

程序支持最小化到系统托盘：

- **最小化窗口**: 点击最小化按钮，程序会最小化到系统托盘
- **托盘图标右键菜单**:
  - 显示主窗口
  - 退出程序
- **双击托盘图标**: 恢复主窗口

## 自定义秘籍

### 添加秘籍

1. 点击主窗口的 **"➕ 添加秘籍"** 按钮
2. 在弹出的对话框中填写：
   - **游戏类型**: 选择魔兽争霸3或星际争霸1
   - **秘籍代码**: 输入秘籍的实际代码（如 "greedisgood"）
   - **描述**: 简要说明秘籍的作用
   - **分类**: 选择或输入分类名称
   - **启用**: 勾选是否启用该秘籍
3. 点击 **"保存"** 按钮完成添加

### 编辑秘籍

1. 在秘籍列表中选中要编辑的秘籍
2. 点击 **"✏ 编辑"** 按钮
3. 修改秘籍信息
4. 点击 **"保存"** 保存更改

### 删除秘籍

1. 在秘籍列表中选中要删除的秘籍
2. 点击 **"🗑 删除"** 按钮
3. 确认删除操作

### 自定义热键绑定

要自定义热键绑定，需要手动编辑配置文件：

1. 打开 `Data\DefaultHotKeys.json` 文件
2. 按照以下格式添加或修改热键绑定：

```json
{
  "id": 11,
  "cheatCodeId": "wc3_pointbreak",
  "key": "F6",
  "modifiers": 0,
  "description": "魔兽3: pointbreak"
}
```

3. 保存文件并重启应用程序

**参数说明**:
- `id`: 唯一标识符
- `cheatCodeId`: 秘籍ID（从秘籍列表中查看）
- `key`: 键盘按键（F1-F12, NumPad0-NumPad9 等）
- `modifiers`: 组合键修饰符（0=无, 1=Alt, 2=Ctrl, 4=Shift）
- `description`: 描述信息

## 设置选项

打开 **"⚙ 设置"** 窗口可以配置以下选项：

### 通用设置

#### 游戏检测
- **检测间隔**: 设置游戏进程检测的时间间隔（500-5000毫秒）
  - 较短的间隔可以更快检测到游戏，但会占用更多系统资源
  - 推荐值: 1000毫秒

#### 输入设置
- **输入延迟**: 每个字符输入之间的延迟时间（10-200毫秒）
  - 增加延迟可以提高兼容性
  - 如果秘籍输入不稳定，可以尝试增加延迟
  - 推荐值: 50毫秒

### 启动选项

- **开机自动启动**: 勾选后，程序会在 Windows 启动时自动运行
- **启动时最小化到托盘**: 启动后直接最小化到系统托盘

### 系统托盘

- **最小化到托盘而不是任务栏**: 点击最小化按钮时隐藏到托盘
- **关闭窗口时最小化到托盘**: 点击关闭按钮时最小化而不是退出

### 语言

- 当前仅支持简体中文

## 常见问题

### Q1: 程序无法检测到游戏

**解决方法**:
1. 确保游戏正在运行
2. 检查游戏进程名称是否正确（war3.exe 或 StarCraft.exe）
3. 尝试以管理员身份运行 GameCheatHelper
4. 在设置中调整检测间隔

### Q2: 按热键后秘籍没有输入

**解决方法**:
1. 确保游戏窗口处于激活状态
2. 检查热键是否与其他程序冲突
3. 在设置中增加输入延迟（推荐 50-100 毫秒）
4. 查看日志文件（logs文件夹）了解详细错误信息

### Q3: 热键与其他程序冲突

**解决方法**:
1. 编辑 `Data\DefaultHotKeys.json` 文件
2. 修改为其他不冲突的热键
3. 重启应用程序

### Q4: 秘籍输入不完整或乱码

**解决方法**:
1. 在设置中增加输入延迟
2. 确保游戏处于窗口模式而非全屏模式
3. 关闭其他可能干扰键盘输入的程序

### Q5: 程序占用CPU过高

**解决方法**:
1. 在设置中增加游戏检测间隔（如 2000-3000 毫秒）
2. 关闭不需要的后台程序

### Q6: 如何卸载程序

1. 直接删除程序文件夹即可
2. 如果启用了开机自启动，需要在设置中取消勾选后再删除
3. 清理注册表（可选）: `HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Run`

## 技术支持

### 日志文件

程序会自动记录运行日志，日志文件位于：
- 路径: `logs\`
- 当日日志: `yyyy-MM-dd.log`
- 错误日志: `yyyy-MM-dd_error.log`

遇到问题时，请查看日志文件以获取详细错误信息。

### 配置文件

所有配置文件位于：
- 秘籍数据: `Data\DefaultCheats.json`
- 热键绑定: `Data\DefaultHotKeys.json`
- 应用配置: `config.json`（首次运行后生成）
- 日志配置: `NLog.config`
