## 📖 简介

**游戏秘籍助手（GameCheatHelper）** 是一款专为《魔兽争霸3》和《星际争霸1》设计的秘籍快捷输入工具。通过全局热键，您可以在游戏中快速输入秘籍代码，无需手动打字，让游戏体验更加便捷！

### ✨ 主要特性

- 🎮 **自动游戏检测** - 自动检测魔兽争霸3和星际争霸1游戏进程
- ⌨️ **全局热键支持** - 按下快捷键即可快速输入秘籍
- 📝 **丰富秘籍库** - 内置常用秘籍，支持自定义添加
- 🔍 **智能搜索** - 快速查找所需秘籍
- 🌐 **系统托盘** - 支持最小化到系统托盘，不占用任务栏
- ⚙️ **灵活配置** - 输入延迟、开机自启等可自定义

---

## 💻 系统要求

- **操作系统**: Windows 10 (64位) 或 Windows 11
- **运行环境**: 已包含在单文件exe中，无需额外安装
- **支持游戏**: 魔兽争霸3、星际争霸1

## 🚀 快速开始

### 1. 启动程序

- 双击 `GameCheatHelper.exe` 启动程序
- 程序会自动显示主窗口并开始检测游戏

### 2. 启动游戏

- 打开《魔兽争霸3》或《星际争霸1》
- 程序顶部状态栏会显示"检测到 XXX"表示已成功检测

### 3. 使用热键输入秘籍

- 在游戏中按下预设的热键（默认 F1-F5）
- 程序会自动在游戏聊天框中输入对应的秘籍代码

### 4. 查看和管理秘籍

- 在主窗口的秘籍列表中查看所有可用秘籍
- 每个秘籍旁边显示其绑定的热键
- 可以启用/禁用、添加、编辑或删除秘籍

## ⚙️ 设置选项

点击主窗口右上角的 **"⚙ 设置"** 按钮可以配置：

### 游戏检测
- **检测间隔**: 设置游戏进程检测的时间间隔（500-5000毫秒）
  - 推荐值: 1000毫秒

### 输入设置
- **输入延迟**: 每个字符输入之间的延迟时间（10-200毫秒）
  - 推荐值: 50毫秒
  - 如果秘籍输入不稳定，可以增加延迟

### 启动选项
- **开机自动启动**: 勾选后，程序会在 Windows 启动时自动运行
- **启动时最小化到托盘**: 启动后直接最小化到系统托盘

## ❓ 常见问题

### Q1: 程序无法检测到游戏？

**解决方法**:
1. 确保游戏正在运行
2. 确认游戏进程名称是否为 `war3.exe` 或 `StarCraft.exe`
3. 在设置中调整检测间隔
4. 尝试重启程序

### Q2: 按热键后秘籍没有输入？

**解决方法**:
1. 确保游戏窗口处于激活状态
2. 在设置中增加输入延迟（推荐 50-100 毫秒）
3. 检查热键是否与其他程序冲突
4. 确保游戏处于可以输入聊天的状态

### Q3: 热键与其他程序冲突？

**解决方法**:
1. 在设置窗口中更改热键绑定
2. 选择其他不冲突的热键（如 F6-F12）
3. 关闭与热键冲突的其他程序

### Q4: 秘籍输入不完整或乱码？

**解决方法**:
1. 在设置中增加输入延迟（推荐 80-100 毫秒）
2. 确保游戏处于窗口模式
3. 关闭其他可能干扰键盘输入的程序

### Q5: 如何卸载程序？

1. 如果启用了开机自启动，先在设置中取消勾选
2. 直接删除程序文件夹即可
3. 无需清理注册表

## 📁 文件说明

程序目录结构：

```
GameCheatHelper/
├── GameCheatHelper.exe    # 主程序（单文件，包含所有依赖）
├── Data/                  # 数据文件夹
│   ├── DefaultCheats.json      # 秘籍数据
│   └── DefaultHotKeys.json     # 热键绑定
├── NLog.config            # 日志配置
└── logs/                  # 日志文件夹（自动生成）
    └── yyyy-MM-dd.log     # 当日日志
```

**注意**: 请勿删除 `Data` 文件夹，否则程序将无法正常运行。

## 🔒 安全说明

本工具**仅使用游戏官方提供的秘籍功能**，不会：
- ❌ 修改游戏内存
- ❌ 注入游戏进程
- ❌ 修改游戏文件
- ❌ 用于联机对战

本工具通过模拟键盘输入的方式，自动化输入秘籍代码，**完全合法且安全**。
